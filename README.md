# Spotify Discovery Tools

Discover new music through Spotify's genre ecosystem. Two powerful browsers to explore and find playlists.

---

## **Try the Apps**

| App | Description | Link |
|-----|-------------|------|
| **Genre Browser** | 24 main genres, 6,291 subgenres | **[Launch App](https://keeltekool.github.io/spotify-discovery/)** |
| **Artist Browser** | 34 genres, 5,819 artists | **[Launch App](https://keeltekool.github.io/spotify-discovery/artist-browser/)** |

---

## Features

- **Instant Search** - Find any genre or artist as you type
- **One-Click Spotify** - Opens playlists directly in Spotify app (mobile & desktop)
- **No Login Required** - Works immediately, no Spotify account needed to browse
- **Mobile Friendly** - Responsive design works on all devices
- **Fast & Lightweight** - Pure HTML/CSS/JavaScript, no frameworks

## How It Works

1. **Browse** - Explore genres or artists by category
2. **Search** - Type to filter through thousands of options
3. **Click** - Tap any item to open Spotify playlists

## Tech Stack

- Pure HTML, CSS, JavaScript (no frameworks)
- JSON data files for genres and artists
- GitHub Pages hosting
